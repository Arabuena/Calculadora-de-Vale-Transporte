## 📱 PWA (Progressive Web App)

Esta aplicação é um PWA e pode ser instalada em:
- 📱 Smartphones Android
- 📱 iPhones
- 💻 Computadores

### Recursos PWA
- ✅ Funciona offline
- ✅ Instalável no dispositivo
- ✅ Atualizações automáticas
- ✅ Ícone na tela inicial
- ✅ Splash screen personalizada
